# Dotfiles

dotfiles for OSX environment

## Contents

+ zsh configuration
+ vim configuration
+ tmux configuration
+ git configuration
+ Homebrew bins
+ Brew-cask apps
+ pip packages


## Install

1. `git clone https://github.com/tafujita/dotfiles.git ~/.dotfiles`
2. `cd ~/.dotfiles`
3. `./install.sh`
4. manually symlink oh-my-zsh `ln -s ~/.dotfiles/zsh/oh-my-zsh ~/.oh-my-zsh/`
